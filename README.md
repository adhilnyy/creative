# Creative

Creative is a sleek and user-friendly web application built to demonstrate clean UI design, responsiveness, and efficient functionality. Developed as part of an interview task, it highlights best practices in frontend development, ensuring a smooth and engaging user experience.

## Features
- User authentication (Login & Signup)
- Protected Routes
- Post creation and management
- Responsive design with Tailwind CSS
- Optimized performance using React and Redux
- Custom hooks for form validation and authentication
- Modal confirmation dialogs

## Tech Stack
- React.js, Tailwind CSS, React Router, Redux Toolkit, Vite

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/adhilnyy/creative.git
   ```

2. Navigate to the project directory:
   ```bash
   cd creative
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```
```

## Usage
- log in to access features.
- Create, update, and delete posts.
- Utilize the interactive UI with modals and validation.
